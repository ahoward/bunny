Run `bny proposal $ARGUMENTS` and report the result.

Generates proposals from the brane or accepts them into the roadmap.

Common usage:
- `bny proposal "topic"` — generate proposal file(s)
- `bny proposal --count 3 "topic"` — generate 3 proposals
- `bny proposal accept <slug>` — accept proposal into roadmap
- `bny proposal --dry-run "topic"` — preview prompt

Show the result to the user.
