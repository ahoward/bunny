Run `bny brane loop $ARGUMENTS` from the project root and report the result.

Autonomous goal-directed thought loop. Iteratively reflects on the worldview,
searches the web, fetches sources, and journals each round.

Common usage:
- `bny brane loop "topic"` — start new loop (1 round)
- `bny brane loop --rounds 5 "topic"` — run 5 rounds
- `bny brane loop --yes "topic"` — auto-incorporate into worldview
- `bny brane loop --propose "topic"` — generate proposals after loop
- `bny brane loop --resume <slug>` — resume existing loop
- `bny brane loop list` — show all loops
