Run `bny brane rebuild $ARGUMENTS` and report the result.

Clears the worldview and re-eats all tracked sources through current lenses.
Rebuilds the brane from scratch.

Common usage:
- `bny brane rebuild` — rebuild worldview
- `bny brane rebuild --dry-run` — show what would be re-eaten
- `bny brane rebuild --yes` — skip confirmation

Show the result to the user.
