Run `bny brane lens $ARGUMENTS` and report the result.

Manage worldview lenses — perspectives through which the brane filters information.

Common usage:
- `bny brane lens` — list all lenses
- `bny brane lens add <name> <desc>` — create a new lens
- `bny brane lens on <name>` — activate a lens
- `bny brane lens off <name>` — deactivate a lens
- `bny brane lens rm <name>` — delete a lens

Show the output to the user.
