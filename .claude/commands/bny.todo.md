Run `bny todo $ARGUMENTS` and report the result.

Manages project todos in `.bny/todos.md`.

Common usage:
- `bny todo` — list todos
- `bny todo add "..."` — add a todo
- `bny todo done N` — mark done
- `bny todo rm N` — remove
- `bny todo promote N` — escalate to GitHub issue

Show the result to the user.
