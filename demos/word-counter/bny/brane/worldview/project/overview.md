# Word Counter CLI
A command-line tool that counts words, lines, and characters in files.

## Core Capabilities

- **Word count** — total words in a file
- **Line count** — total lines in a file
- **Character count** — total characters in a file

## Nature

This is a text analysis utility operating on files provided as input. It follows the classic Unix tradition of simple, composable CLI tools that do one thing well.
