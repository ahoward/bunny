# Bunny

Bun + TypeScript project with spec-kit and DNA systems.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev       # Run
bun test          # Test
bun run build     # Compile binary
```

## Workflow

See `dna/product/ROADMAP.md` for the driving task list.

Uses spec-driven development: ROADMAP → `/speckit.specify` → spec PR → `/speckit.plan` → `/speckit.tasks` → implement.
